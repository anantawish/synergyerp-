# SynergyERP Service Offer

## Positioning

SynergyERP is the public proof of capability. The paid business is installation,
customization, data migration, maintenance, and AI that understands each client's
operation.

## Services

### Setup Package

- install on XAMPP, VPS, shared hosting, or internal server
- configure database and environment variables
- restore demo or client database
- create admin users and departments
- basic hardening and backup schedule

### Custom ERP Package

- map existing Excel or legacy system workflows
- customize sales, purchase, stock, accounting, HR, and manufacturing modules
- create custom forms, approvals, reports, and print templates
- import master data and transaction history
- build dashboards for owners and managers

### Maintenance Package

- monthly update window
- backup review and restore test
- issue triage and bug fixes
- performance checks
- user support and small improvements

### RAG AI Add-on

- connect user manuals, SOPs, contracts, invoices, reports, and database views
- let staff ask questions about stock, sales, purchase, accounting, HR, and
  production
- cite manuals, reports, or records used in the answer
- apply role/department permissions
- keep production data inside the client environment

## Sales Message

Use the public ERP to prove the foundation. Then sell the implementation layer:
installation, migration, custom workflow, monthly care, and AI that can answer
questions from the customer's own business data.
