Daily Checklist

Start of Day

  1. Check previous-day unposted journal and open reconciliation items.
  2. Validate AP/AR cash movements expected for today.
  3. Confirm document sequence and posting date controls.

Operations

  1. Post journals from sales, purchase, inventory, and payroll events.
  2. Record VAT/WHT and verify taxable base against source docs.
  3. Settle AP/AR and tie each payment to billing reference.
  4. Investigate and resolve out-of-balance entries immediately.

Close of Day

  1. Run Trial Balance and verify debit=credit.
  2. Run Balance Sheet snapshot and compare with prior close.
  3. Archive report and evidence set for audit trail.

Audit Evidence To Capture

  1. Posted GL journal with source_ref and line details.
  2. VAT/WHT summary report for period checkpoint.
  3. Daily balance sheet snapshot.
