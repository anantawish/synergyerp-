1) Before Start

  1. Sign in at index.php and verify permission scope from department_access.php or admin_access.php.
  2. Confirm core master data: items, customers/suppliers, chart of accounts, BOM/routing, warehouse mapping.
  3. Prepare references for traceability: project code, document number, lot number, and run number.
  4. Capture critical evidence using capture_screen.php during the process.

4) Capture Module (Audit Evidence)

  1. Open capture_screen.php and choose the relevant module_key.
  2. Fill screen_name, process_stage, project_code/run_no/doc_ref.
  3. Upload screenshot image (PNG/JPG/WEBP/GIF/BMP, max 15MB).
  4. System writes capture_no and stores file under storage/captures/YYYY/MM/.
  5. Deletion is soft-delete and remains traceable in app_soft_delete.
